Os seletores de classe permitem aplicar estilos a um ou mais elementos que possuem uma determinada característica. Para definir uma classe em um elemento HTML, utiliza-se o atributo `class` seguido do nome da classe. No CSS, as classes são selecionadas utilizando um ponto (`.`) seguido do nome da classe.
